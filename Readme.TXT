В этот раз задача состоит в реализации двух разновидностей сортировки Хоара (называемой также быстрой, quicksort) с различным выбором опорного элемента.
Решить данную задачу требуется по классической схеме Хоара (с двумя индексами).

В частности, в данной задаче необходимо выбирать в качестве опорного средний элемент.

Формат ввода
В первой строке — количество элементов во входном массиве.
Во второй строке — массив с целыми элементами (со знаком) через пробел.

Формат вывода
Одна строка — отсортированный массив с элементами через пробел от меньшего к большему.
